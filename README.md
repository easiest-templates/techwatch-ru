# TechWatch

Куратированная подборка лучших технических докладов и конференций для разработчиков.

![Preview](https://cdn.poehali.dev/templates/techwatch-ru/preview-ru.jpg)

## Технологии

- React 18
- TypeScript
- Tailwind CSS
- Vite
- React Router
- React Query
- Radix UI
- Lucide Icons

## Установка

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

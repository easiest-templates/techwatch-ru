# TechWatch — Лента технических видео

![Preview](https://cdn.poehali.dev/templates/techwatch-ru/preview-ru.jpg)

Это шаблон сайта с подборкой технических видео и докладов для разработчиков. Выполнен в бруталистском стиле — минимум украшений, максимум функциональности. Такой дизайн отлично подходит для IT-аудитории, которая ценит содержание больше формы.

На сайте собраны видео с конференций и технических митапов. Каждое видео представлено карточкой с названием, описанием и превью. Можно фильтровать видео по категориям и искать по ключевым словам.

Шаблон подойдет для создания каталога обучающих видео, архива докладов с конференций или персональной подборки полезных материалов. Дизайн выглядит технично и профессионально — именно то, что нужно для IT-проектов.

Все тексты уже переведены на русский язык. Просто добавьте свои видео и настройте категории под ваши нужды.
